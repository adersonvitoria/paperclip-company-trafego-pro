Os 5 pipelines

Fluxos completos coordenados pelo CEO — cada etapa é executada pelo agente dono da skill.

1

Lançamento de Campanha · 45–90 min

"Quero lançar tráfego pago / campanha nova no Google Ads"

keyword-research → competitor-recon → media-plan-builder → tracking-blueprint → lp-cro-audit → ad-copy-builder → builders (Search/PMax/Vídeo) → ⛔ GATE account-audit
2

Otimização Semanal · 20–40 min

"Roda a otimização da semana / revisa as campanhas"

performance-report → optimization-routine → budget-pacing
3

Auditoria 360 da Conta · 40–60 min

"Audita minha conta / por que meu Google Ads não performa?"

account-audit → tracking-blueprint (verificação) → lp-cro-audit → media-plan-builder (plano de correção)
4

Escala de Performance · 30–50 min

"Campanha está boa, quero escalar"

performance-report → media-plan-builder (escala) → budget-pacing → builders → gads-scripts → ⛔ GATE account-audit
5

Relatório Mensal Executivo · 15–25 min

"Fecha o mês / relatório pro cliente"

performance-report (mensal) → media-plan-builder (recomendações do próximo mês)
⛔ Invariante da casa: nenhuma campanha sobe e nenhuma verba é escalada sem o veredito PASS do Auditor de Conta & Compliance. A auditoria é gate obrigatório, não etapa opcional.

Como usar

Este é um Agent Company Package do Paperclip — configuração pura (markdown), sem build.

1. Clone o repositório e importe a empresa:

paperclipai company import ./paperclip-company-trafego-pro

2. Abra uma issue em linguagem natural — ex.: "quero lançar tráfego pago para minha clínica em Vitória/ES com R$ 3.000/mês" — e o CEO roteia o pipeline.
